#  Sistem Pencarian Berbasis TF-IDF

Proyek ini adalah aplikasi web yang mengimplementasikan metode **TF-IDF (Term Frequency - Inverse Document Frequency)** untuk melakukan pencarian teks atau dokumen secara relevan.

##  Fitur Utama
* **Tekstual Search**: Pencarian dokumen berdasarkan kata kunci (keyword).
* **Pembobotan TF-IDF**: Menghitung bobot term pada setiap dokumen secara dinamis.
* **Perankingan Relevansi**: Menampilkan hasil pencarian dari yang paling relevan hingga kurang relevan.

##  Teknologi yang Digunakan
* **Bahasa Pemrograman**: PHP
* **Database**: MySQL
* **Environment**: Laragon / XAMPP

##  Cara Menjalankan Proyek

1. Unduh file ZIP-nya.
2. Pindahkan folder proyek ke direktori server lokal Anda (misal: `C:\laragon\www\` atau `C:\xampp\htdocs\`).
3. Import database `nama_database.sql` (jika ada) ke phpMyAdmin.
4. Sesuaikan konfigurasi database pada file koneksi proyek.
5. Buka browser dan akses `http://localhost/searching-tf-idf-master`.
